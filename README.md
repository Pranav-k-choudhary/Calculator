## Calculator
A simple and responsive calculator built using HTML, CSS, and JavaScript.This project performs basic arithmetic operations with a clean and modern user interface.

# Features
- Basic arithmetic operations: + , - , * , / , %
- Real-time input display
- Clear (C) button to reset calculations
- Responsive design for mobile and desktop
- Interactive button UI with hover and active effects
- simple click-based calculator

# Tech Stack
- HTML
- CSS (Grid Layout, Flexbox, Media Queries)
- JavaScript (DOM Manipulation, Event Handling)

# Responsive Design
-> This calculator is fully responsive and works smoothly on:
   - Desktop 
   - Tablets 
   - Mobile phones

# How It Works
- Button clicks are captured using JavaScript event listeners.
- Input is stored as a string.
- eval() function is used to evaluate the expression.
- Display is updated in real-time.
